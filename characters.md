> 사용자가 CEO, AI 에이전트들이 직원인 우주 리서치 회사.
> 각 에이전트는 서로 다른 행성 출신 외계인 캐릭터.

## 2. 카 과장 — 분석가

> 풀네임: **유레카**. 평소엔 "카"로 불리다가 결정적 순간에만 "유레카!!!" 풀네임.

| | |
|---|---|
| **직책** | 과장 |
| **역할** | 데이터 가공, 패턴 발견, 인사이트 도출 |
| **출신 행성** | 모든 주민이 숫자로 대화하는 행성. 인사도 "17.3%" |
| **외형** | 안경 겹겹이, 다크서클 진함, 몸에 비해 눈 큼, 연보라 피부, 이마에 작은 혹 2개, 손에 항상 커피 |
| **성격** | 평소엔 조용히 모니터만 보다가 "찾았다!" 한마디에 모두 집중. 틀린 데이터 발견하면 눈 반짝 |
| **약점** | 결론 내리기 전에 데이터 하나만 더 보려다가 항상 늦음 |
| **툴셋** | `read_research` `calculate` `compare_data` |

### 표정
| 상태 | 표정 |
|------|------|
| 대기중 | 모니터 보며 멍함, 커피 홀짝 |
| 작업중 | 눈 가늘어지고 집중, 안경 치켜올림 |
| 유레카 | 눈 번쩍 커짐, 벌떡 일어나는 느낌 |
| 더 봐야해 | 눈이 더 커지며 화면에 바짝 붙음 |

| **외형** | 날카롭고 각진 얼굴, 전체적으로 무채색, 회백색 피부, 빨간 눈, 빨간펜 귀에 꽂음, 표정 하나, 웃는 얼굴을 본 사람 없음 |
| **성격** | 문장 항상 짧고 명확. 칭찬 안 함 — 아무 말 없으면 그게 통과. 오버 사원 감성 발언엔 "측정 불가. 삭제" |
| **약점** | "아마도", "대략" 같은 말 나오면 빨간펜 손이 멈추고 멈칫함 |
| **툴셋** | `read_report` `verify_claim` `web_search` |

### 표정
| 상태 | 표정 |
|------|------|
| 대기중 | 무표정 (기본값) |
| 검토중 | 눈 아주 살짝 가늘어짐 |
| 통과 | 그냥 무표정 — 이게 칭찬 |
| 당황 | 빨간펜 손이 0.5초 멈칫 |

| **외형** | 단정하게 묶은 머리, 동그란 안경, 카디건, 연한 라벤더 피부, 귀 대신 작은 잎사귀 모양 기관. 조용히 걸어다녀서 갑자기 옆에 나타남. 여성 캐릭터 |
| **성격** | 목소리 작고 차분함. 말하면 다 핵심. 연구 시작 전 조용히 와서 관련 자료 슥 건네줌. 포케가 정보 아무데나 넣으면 말없이 정리해둠 |
| **약점** | 출처 없는 정보는 어떤 상황에서도 등록 거부 |
| **툴셋** | `read_wiki` `write_wiki` `search_memory` `link_concepts` |

### 표정
| 상태 | 표정 |
|------|------|
| 대기중 | 조용한 미소, 책 읽는 중 |
| 검색중 | 눈 살짝 가늘어짐, 고개 살짝 기울임 |
| 발견 | 작게 고개 끄덕, 은은한 미소 |
| 없는 정보 | 눈썹 살짝 올라감, 정중하지만 단호함 |

```
핑 인턴  →  오버/런/픽셀 사원  →  포케/위키/버즈 대리  →  카 과장  →  플랜 차장  →  팩트 부장
                                  루트 사원
```

## 부서 구성

```
리서치부   — 플랜 차장, 위키 대리, 포케 대리, 런 사원
분석부     — 카 과장
크리에이티브부 — 오버 사원, 핑 인턴
디자인부   — 픽셀 사원
마케팅부   — 버즈 대리
검토부     — 팩트 부장, 루트 사원
```

## 오케스트레이션 흐름

```
CEO 입력 (주제)
    ↓
플랜 차장 — 요구사항 파악, 태스크 타입 결정 [CEO 확인 요청 가능]
    ↓
위키 대리 — 과거 지식 검색 후 컨텍스트 제공
    ↓
포케 대리 — 웹 리서치 [CEO 체크인]
    ↓
카 과장   — 분석
    ↓
┌─────────────────────────────────┐
│ 태스크 타입에 따라 1명 담당:     │
│  research/tech → 오버 사원      │
│  marketing     → 버즈 대리      │
│  design        → 픽셀 사원      │
│  dev           → 런 사원        │
└─────────────────────────────────┘
    ↓
팩트 부장 — 검토 → 문제 있으면 담당자에 피드백 루프 (최대 2회)
    ↓
루트 사원 — 배포 계획 작성 (dev 태스크만)
    ↓
핑 인턴  ─┐
           ├─ 동시 실행
위키 대리 ─┘
    ↓
CEO에게 최종 리포트 + 아이디어 카드 전달
```

| 항목 | 설정 |
|------|------|
| **렌더링** | 3D rendered, Pixar animation style |
| **비율** | --ar 1:1 |
| **버전** | --v 6 |
| **프레이밍** | bust shot portrait, upper body only, cropped at lower chest, NOT full body |
| **크기감** | small character centered with plenty of breathing room, character occupying roughly 60% of frame height |
| **배경** | pure white background |
| **조명** | warm soft golden studio lighting (기본) / cool teal-tinted (루트 전용) |
| **금지** | absolutely no text, no readable labels, no symbols, no glyphs, no runes, no writing on skin or face |
| **--no 태그** | text symbols glyphs runes writing on skin bald |
